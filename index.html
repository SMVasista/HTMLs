Welcome to RR Turbo - The Interfaced Accelerator for RR tools

RR is a cellworks(r) centered, auxillary development platform comprising of 45+ (and counting) modeling and quality check specific tools to make the overall process automatic, hassel-free and stream-lined for development pipelines. You can access all the below features and generate/perform specific functions as needed. To know more about RR click here


Available Modules

All the available modules are listed in the RR shell terminal and can be accessed by available_modules command. Here we list the set of preset chain of commands that are used in everyday development progression. These set of commands can be chained in the form of a batch and run from RR to extract results in shorter time.
